# Workpiece-Dataset

This workpiece dataset contains 748 point cloud models of pipe bends with various surface protrusion defects.

# Data Organization

The data is organized in the following format:

```text
/workpiece3D/
    └── sequences/
        ├── fusion1.txt
        ├── fusion2.txt
        ├── fusion3.txt
        .
        .
        .
        └── fusion748.txt
```
